## 🚀 如何运行

> node 版本 `[12.13.3]`

### ✨ 开发过程

```
# 克隆
git clone https://github.com/ZanHaitao/VUE-Meituan.git
```

```
# 打开项目目录
cd vue-meituan
```

```
# 安装依赖
npm install
```

```
# 开启本地服务运行项目
npm run serve
```

```
# 常用命令
npm run serve // 运行项目
npm run build // 打包项目
```
